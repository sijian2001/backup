バックアップ用のシェルを作ってください。
ファイル名：backup.sh
引数1: dir フォルダ名
引数2: mode 1:年 2:年月

処理内容：
1.引数チェック
　引数設定されていること
　引数1のフォルダは存在すること
　引数2設定されて、1 or 2であること
　　
２．データバックアップ
　引数のフォルダは以下に移動する
  引数2は1:年の場合、バックアップフォルダは2001-2024

　年月（202501から202508まで）を繰り返して、下の処理をする
　　年月.zipフォルダ存在する場合、次の年月を処理する
　　年月.zipフォルダ存在しない場合
　　　　年月のフォルダを作る
　　　　ファイル名が「*年月*」のものを年月フォルダに移動する
　　　　年月のフォルダを圧縮して、年月.zipを作成する。
　　　　圧縮は正常終了の場合、年月のフォルダを削除する。
その他
　処理する時、処理状況を分かるように、コンソールに情報を出力してください。
